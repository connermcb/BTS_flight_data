List of Resources

American Statistical Association - source of subsetted data

Bureau of Transportation Statists - original source of data
https://www.bts.gov/

http://stat-computing.org/dataexpo/2009/carriers.csv

github user repository  - source of airport IATA codes, names, and variables
https://raw.githubusercontent.com/datasets/airport-codes/master/data/airport-codes.csv
